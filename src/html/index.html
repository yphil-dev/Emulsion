Getting Started

  • Pick a platform / console from the home screen or the settings menu ;
  • Select the games folder for that platform ;
  • Set your preferred emulator executable for that platform ;
  • (Optional) Add custom launch arguments to fine-tune how each emulator runs.

Using Emulsion

  • Browse your game library by cover or title
  • Optionnaly get the artwork and metadata of either one game or all the games in the current platform
  • Press Enter (or your game controller's fire button) to start the game üöÄ
